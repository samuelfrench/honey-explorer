# Plan 04-01: Tailwind Configuration and Color Palette

## Objective
Configure Tailwind CSS v4 with a honey-themed design system including custom colors, typography, and spacing tokens.

## Context
- Tailwind v4 uses CSS-first configuration (no tailwind.config.js)
- Current setup is minimal: just `@import "tailwindcss";` in index.css
- Need honey color palette: amber, gold, warm browns
- Need design tokens for consistency

## Tasks

### 1. Create CSS-based Tailwind Configuration
**File**: `frontend/src/index.css`

Replace minimal import with full theme configuration:

```css
@import "tailwindcss";

/* Honey Explorer Theme */
@theme {
  /* Honey Color Palette */
  --color-honey-50: #fffbeb;
  --color-honey-100: #fef3c7;
  --color-honey-200: #fde68a;
  --color-honey-300: #fcd34d;
  --color-honey-400: #fbbf24;
  --color-honey-500: #f59e0b;
  --color-honey-600: #d97706;
  --color-honey-700: #b45309;
  --color-honey-800: #92400e;
  --color-honey-900: #78350f;
  --color-honey-950: #451a03;

  /* Warm Browns (Comb) */
  --color-comb-50: #fdf8f6;
  --color-comb-100: #f2e8e5;
  --color-comb-200: #eaddd7;
  --color-comb-300: #e0cec7;
  --color-comb-400: #d2bab0;
  --color-comb-500: #bfa094;
  --color-comb-600: #a18072;
  --color-comb-700: #977669;
  --color-comb-800: #846358;
  --color-comb-900: #43302b;

  /* Cream Background */
  --color-cream: #fffef9;
  --color-cream-dark: #faf8f0;

  /* Typography - Warm readable fonts */
  --font-sans: "Inter", ui-sans-serif, system-ui, sans-serif;
  --font-display: "Playfair Display", ui-serif, Georgia, serif;

  /* Shadows - Warm tinted */
  --shadow-honey-sm: 0 1px 2px 0 rgb(180 83 9 / 0.05);
  --shadow-honey: 0 1px 3px 0 rgb(180 83 9 / 0.1), 0 1px 2px -1px rgb(180 83 9 / 0.1);
  --shadow-honey-md: 0 4px 6px -1px rgb(180 83 9 / 0.1), 0 2px 4px -2px rgb(180 83 9 / 0.1);
  --shadow-honey-lg: 0 10px 15px -3px rgb(180 83 9 / 0.1), 0 4px 6px -4px rgb(180 83 9 / 0.1);

  /* Border Radius */
  --radius-card: 1rem;
  --radius-button: 0.5rem;
  --radius-badge: 9999px;

  /* Spacing (extending defaults) */
  --spacing-card: 1.5rem;
  --spacing-section: 4rem;
}

/* Base layer customizations */
@layer base {
  html {
    @apply antialiased;
  }

  body {
    @apply bg-cream text-comb-900;
  }

  h1, h2, h3 {
    @apply font-display;
  }
}
```

### 2. Add Google Fonts
**File**: `frontend/index.html`

Add font links in `<head>`:
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=Playfair+Display:wght@400;500;600;700&display=swap" rel="stylesheet">
```

### 3. Update App.tsx with New Colors
Replace amber-* classes with honey-* classes to test the configuration:
- `amber-50` → `honey-50`
- `amber-500` → `honey-500`
- `amber-700` → `honey-700`
- `amber-800` → `honey-800`

### 4. Verify Configuration Works
- Run `npm run dev`
- Confirm colors render correctly
- Confirm fonts load
