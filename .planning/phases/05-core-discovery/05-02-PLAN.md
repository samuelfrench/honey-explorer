# Plan 05-02: Browse/Filter Interface with Faceted Search

## Objective
Build the honey browse page with search, filters, and large visual grid of honey products.

## Context
- Filter options API exists from Phase 2
- 210 honeys with images
- Need instant search + multi-facet filtering
- Large product imagery (not tiny thumbnails)

## Tasks

### 1. Create Browse API with Filtering
**File**: `backend/.../HoneyController.java`

```java
@GetMapping
public Page<HoneyDTO> browse(
    @RequestParam(required = false) String search,
    @RequestParam(required = false) List<String> origin,
    @RequestParam(required = false) List<String> floralSource,
    @RequestParam(required = false) List<String> honeyType,
    @RequestParam(required = false) List<String> flavorProfile,
    @RequestParam(defaultValue = "0") int page,
    @RequestParam(defaultValue = "24") int size,
    @RequestParam(defaultValue = "name") String sort
) {
    return honeyService.browse(search, origin, floralSource, honeyType, flavorProfile, page, size, sort);
}
```

### 2. Build HoneyRepository Query Methods
