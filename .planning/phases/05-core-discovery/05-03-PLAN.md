# Plan 05-03: Honey Detail Page and Comparison

## Objective
Build the honey detail page with full-width hero image, info cards, and comparison feature.

## Context
- Honeys have rich metadata (flavor profiles, pairings, certifications)
- Visual-first approach: hero image, info cards
- Comparison feature for side-by-side view

## Tasks

### 1. Create Honey Detail API
**File**: `backend/.../HoneyController.java`

```java
@GetMapping("/{slug}")
public HoneyDetailDTO getBySlug(@PathVariable String slug) {
    return honeyService.getBySlug(slug);
}
```

DTO includes:
- All honey fields
- Related honeys (same floral source)
- Similar honeys (by flavor profile)

### 2. Build HoneyDetailPage
**File**: `frontend/src/pages/HoneyDetailPage.tsx`

Sections:
1. **Hero Image** - Full-width, 16:9 or 3:2 aspect
2. **Title Bar** - Name, origin badge, floral source badge
3. **Quick Facts** - Texture, color intensity, honey type
4. **Description** - Rich text description
5. **Flavor Profile** - Visual tags/badges
6. **Certifications** - UMF/MGO badges with tooltips
7. **Pairings** - Visual icons (cheese, tea, toast)
8. **Related Honeys** - Similar items carousel

### 3. Create CertificationBadge Component
**File**: `frontend/src/components/honey/CertificationBadge.tsx`

- UMF rating display
- MGO rating display
- Tooltip on tap/hover explaining certification
- Visual differentiation by rating level

### 4. Create PairingIcons Component
**File**: `frontend/src/components/honey/PairingIcons.tsx`

Visual icons for:
- Cheese
- Tea
- Toast/Bread
- Yogurt
- Fruit
- Meat (glazes)

### 5. Create FlavorProfileVisual Component
**File**: `frontend/src/components/honey/FlavorProfileVisual.tsx`

- Visual representation of flavor notes
- Color-coded badges
- Intensity indicators

### 6. Build Comparison Feature
**File**: `frontend/src/pages/ComparePage.tsx`

- Add to compare button on cards
- Compare drawer/modal
- Side-by-side comparison table
- Visual diff highlighting
- Max 2 honeys for comparison

### 7. Create CompareButton Component
**File**: `frontend/src/components/honey/CompareButton.tsx`

- Toggle add/remove from comparison
- Badge showing comparison count
- Link to comparison page when 2 selected

### 8. Implement Compare State
**File**: `frontend/src/hooks/useCompare.ts`

- localStorage persistence
- Max 2 items
- Add/remove/clear functions
