# Plan 05-04: SEO and Meta Tags

## Objective
Implement SEO meta tags and schema.org markup for honey discovery pages.

## Context
- Need search engine visibility
- Open Graph tags for social sharing
- Schema.org Product markup
- Dynamic meta based on page content

## Tasks

### 1. Install React Helmet Async
```bash
npm install react-helmet-async
```

Configure provider in main.tsx.

### 2. Create SEO Component
**File**: `frontend/src/components/seo/SEO.tsx`

```tsx
interface SEOProps {
  title: string;
  description: string;
  image?: string;
  url?: string;
  type?: 'website' | 'product';
  schema?: object;
}
```

Renders:
- `<title>`
- `<meta name="description">`
- Open Graph tags (og:title, og:description, og:image, og:url)
- Twitter Card tags
- Canonical URL
- JSON-LD schema

### 3. Homepage SEO
- Title: "Honey Explorer - Discover Amazing Honey from Around the World"
- Description: "Browse 200+ honey varieties, find local beekeepers..."
- Schema: WebSite with SearchAction

### 4. Browse Page SEO
- Dynamic title based on filters
- Description: "Browse and filter honey by origin, floral source..."
- Schema: ItemList

### 5. Honey Detail Page SEO
- Title: "{Honey Name} - Honey Explorer"
- Description: First 160 chars of honey description
- Image: Honey image URL
- Schema: Product with offers, aggregateRating

### 6. Create Schema Generators
**File**: `frontend/src/utils/schema.ts`

Functions to generate:
- WebSite schema
- Product schema
- ItemList schema
- BreadcrumbList schema

### 7. Add Robots.txt
**File**: `frontend/public/robots.txt`

```
User-agent: *
Allow: /
Sitemap: https://honey-explorer.fly.dev/sitemap.xml
```

### 8. Generate Sitemap
Backend endpoint or build-time generation:
- Homepage
- Browse page
- All honey detail pages
- All local source pages

## Success Criteria
- [ ] Meta tags render correctly
- [ ] Open Graph tags work (test with social debuggers)
- [ ] Schema.org JSON-LD validates
- [ ] Sitemap accessible
- [ ] Robots.txt in place

## Dependencies
- 05-01 to 05-03 (pages to add SEO to)

## Estimated Complexity
Small-Medium - mostly configuration
